ActionScript 3.0 中有两种函数类型：方法和函数闭包。将函数称为方法还是函数闭包取决于定义函数的上下文。

方法

如果您将函数定义为类定义的一部分或者将它附加到对象的实例，则该函数称为方法。

闭包

如果您以其它任何方式定义函数，则该函数称为函数闭包。闭包，指的是词法表示包括不被计算的变量的函数，也就是说，函数可以使用函数之外定义的变量。就像使用任何高级函数一样，使用闭包要小心，因为它们可能会变得非常复杂。 同时有两种函数定义方法: 一种是非常熟悉的函数语句（Function statement）定义法； 
function  函数名(参数 1:参数类型,参数 2:参数类型...):返回值类型  { 
	//函数内部语句
} 
//例子：
functiontestAdd(a:int, b:int):int { 
return a+b; 
}
另一种是 ActionScript 特有的函数表达式（Function expression）定义法。 
var  函数名:Function =function(参数 1:参数类型,参数 2:参数型...) :返回值类型  { 
		//函数内部语句
} 
//例子：
var testAdd:Function =function(a:int,b:int):int { 
	return a+b; 
}
